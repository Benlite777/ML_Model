

# Auto ML + AI Insight App

An intelligent AutoML and AI Insights application built with Streamlit and Google Gemini. Upload your CSV data, train multiple machine learning models, compare results, and get actionable AI-generated insights and suggestions.

---

## 🚀 Features

- 📂 Upload CSV datasets
- ✂️ Automatic data preprocessing (missing value handling, encoding, scaling)
- 🧠 Problem type detection (classification/regression)
- ⚡ Model training: Linear/Logistic Regression, Random Forest, Gradient Boosting
- 📊 Performance comparison with metrics and charts
- 🤖 AI-generated insights and improvement suggestions (Gemini LLM)
- 💬 Interactive Streamlit UI
- ⬇️ Downloadable results

---

## 🛠️ Tech Stack

- **Frontend:** Streamlit
- **LLM:** Google Gemini (`google-generativeai`)
- **ML Framework:** scikit-learn
- **Data Processing:** pandas, numpy

---

## 📌 How It Works

1. User uploads a CSV file
2. Data is automatically preprocessed (missing values, encoding, scaling)
3. User selects the target column
4. App detects if the problem is classification or regression
5. Multiple models are trained and evaluated
6. Results are displayed with metrics and charts
7. User can generate AI insights and suggestions using Gemini

---

## 💡 Usage

1. Upload a CSV dataset
2. Select the target column
3. Wait for processing and model training
4. View model results and charts
5. Click to generate AI insights or suggestions
6. Download results as CSV
